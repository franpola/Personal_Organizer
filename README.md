# Wanderlog 🧭

Personal travel & events planner. Track trips, urban events, and reflect on each one with a post-plan debrief.

## Features

- **Trips** — flights, accommodation, budget tracker, map
- **Events** — urban events (Meetup, concerts, exhibitions), how to get there
- **Debrief** — after each plan, rate it and write your impressions
- Upcoming vs past separation, auto-detected by date
- Persistent via localStorage (no backend needed)

## Stack

- React + Vite
- React Router v6
- lucide-react icons
- Google Maps Embed (no API key needed for basic embed)
- Deployed on Vercel

## Dev

```bash
npm install
npm run dev
```

## Deploy

1. Push to GitHub
2. Import repo in Vercel
3. Framework: Vite — no config needed
4. Deploy ✓
